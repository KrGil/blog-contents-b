## Code convention
### Folder
영문 소문자로만 구성한다.
가능하면 짧게 구성한다(축약어 사용).
특수문자와 빈 공백sᴘᴀᴄᴇ은 사용하지 않는다.
단어와 단어의 구분은 ‘-’(ʜʏᴘʜᴇɴ)으로 구성한다.

### File
가능하면 영문 소문자로만 구성한다.
가능하면 짧게 구성한다(축약어 사용).
특수문자와 빈 공백sᴘᴀᴄᴇ은 사용하지 않는다.
단어와 단어의 구분은 ‘_’(ᴜɴᴅᴇʀʙᴀʀ) 로 구성한다.
